# Minecraft Java Subdomains Guide

{: .warning}
Subdomains are currently available for **Minecraft Java** servers only.

{: .warning}
Each server currently supports **one subdomain**.

---

## Creating a Subdomain

1. Log in to the [ApexNode panel](https://panel.apexnode.host)
2. Go to your Minecraft Java server.
3. Click the **Subdomains** tab.
4. Click **Create Subdomain**.
5. Enter a unique name for your subdomain.
6. Select a domain from the dropdown (if applicable).
7. Choose the allocation you want the subdomain to point to.
8. Click **Create Subdomain** to finish.

---

## Updating an Existing Subdomain

1. Click **Update** next to the subdomain you want to change.
2. Select the new allocation you want it to point to.
3. Click **Update Subdomain** to save changes.
